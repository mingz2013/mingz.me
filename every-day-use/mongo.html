1. mongo

1.1. use in macos

  • brew install mongo
  • brew services start mongodb
  • brew services stop mongodb
  • mongod --config /usr/local/etc/mongod.conf

1.2. use in centos 7

  • vim /etc/yum.repos.d/mongodb-org-4.0.repo ```repo [mongodb-org-4.0] name=MongoDB Repository baseurl=https://repo.mongodb.org/yum/redhat/$releasever/mongodb-org/4.0/x86_64/ gpgcheck=1 enabled=1 gpgkey=https://www.mongodb.org/static/pgp/server-4.0.asc

- `yum install mongodb-org`
- `systemctl restart mongod.service`
