

# go-micro



## 概述

> Micro是一套微服务构建工具库。对于微服务架构的应用，Micro提供平台层面、高度弹性的工具组件，让服务开发者们可以把复杂的分布式系统以简单的方式构建起来，并且尽可能让开发者使用最少的时间完成基础架构的构建。


### 特性
- 网关
- Web Dashboard

- 服务发现
- 负载均衡
- 同步通信
- 异步通信
- 消息编码
- 服务接口-Golang开发框架

- 组件可插拔
