# 基本操作
- command + shit + F   先选中摄像机，将场景所见变成摄像机视角
- q w e r t y    快捷键
- alt + 滑动  改变视角
- ctrl + 滑动 缩放场景
- command + d  复制一个对象
- 按住command拖动，一米一米的拖动
